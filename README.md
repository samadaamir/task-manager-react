## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/samadaamir/task-manager-react.git
```

Navigate into the project

```bash
cd task-manager
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
